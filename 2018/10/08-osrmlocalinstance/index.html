创建本地OSRM实例，用OSM做导航数据

发表于 2018-10-08 | 分类于 开发 Development ， Open Street Map (OSM) |

介绍

Open Source Routing Machine (OSRM) 是一个导航路径计算库。
给定两个地理坐标，通过OSRM可以计算其间的路径、距离、交通时间，并提供 HTTP 和 C++ 的接口。
由于它是一个开源的库，你可以把它的实例安装在本地，以构成一个导航服务。
这样可以避免官方服务器的限额问题。
